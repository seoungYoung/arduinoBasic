1. 단순히 3개의 LED 신호등 형태 동작 : 
	빨강, 노랑, 녹색 LED를 각각 DP 5, 6, 7번에 연결하고, 전원을 연결하면 빨강 5초, 노랑 0.5초 두번 점멸, 이후 녹색 7초 켜짐 동작이 무한 반복하도록 한다.

2. 한 개 버튼 : 풀업 방식으로 한 개의 LED 제어 
	버튼을 DP 3번에 연결하고, LED는 11번에 풀업방식으로 연결하고, 버튼을 누르면 LED가 꺼지고 놓으면 켜진다.

3. 한 개 버튼 : 풀다운 방식으로 한 개의  LED 제어
	버튼을 DP 3번에 연결하고, LED는 11번에 풀다운 방식으로 연결하고, 버튼을 누르면 LED가 켜지고 놓으면 꺼진다.
4. 2개의 LED를 이용 파도타기 아날로그 동작
	2개의 LED를 아날로그 출력 가능한 핀에 연결하고, 전원을 연결하면 빨강 LED가 점점 밝아지고 어두워지고, 이어서 노랑 LED가 점점 밝아지고 어두워지는 동작이 무한 반복하도록 한다.

5. 한 개 버튼 : 풀업 방식으로 한 개의 아날로그 LED 제어
	1개의 LED를 아날로그 출력 가능한 핀에 연결하고, 버튼을 7번 핀에 풀업 방식으로 연결. 버튼을 놓은 상태에서 LED가 서서히 밝아지고 어두워지는 동작을 지속하며 , 버튼을 누르면 LED동작을 멈춘다.

6. 한 개 버튼 : 풀다운 방식으로 한 개의 아날로그 LED 제어
	1개의 LED를 아날로그 출력 가능한 핀에 연결하고, 버튼을 5번 핀에 풀다운 방식으로 연결. 버튼을 누르면 LED가 서서히 밝아지고 어두워지는 동작으로 지속하며, 버튼을 놓으면 LED동작을 멈춘다.
